## 📌 Project Overview

**Smart Spam Detector Pro** is a Machine Learning–based web application that detects whether a message is **Spam** or **Not Spam**.

This project uses **Natural Language Processing (NLP)** and **Machine Learning** to analyze text messages and classify them accurately. The application also provides analytics, prediction history, downloadable reports, and a modern user interface.

---

## ✨ Features

✅ Spam / Not Spam Prediction
✅ Machine Learning Model (Naive Bayes)
✅ Text Preprocessing (Tokenization, Cleaning)
✅ Prediction History Tracking
✅ Download Predictions as CSV
✅ Spam Analytics Graph
✅ Dark Mode UI 🌙
✅ Clean Modern Web Interface
✅ Live Deployment on Render 🌐

3️⃣ Machine Learning model analyzes the message
4️⃣ Prediction is displayed (Spam or Not Spam)
5️⃣ Result is stored in history
6️⃣ Analytics graph updates automatically

## 🚀 Installation (Run Locally)

Clone the repository:

```
git clone https://github.com/yourusername/spam-detector-pro.git
```

Go to project folder:

```
cd spam-detector-pro
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
